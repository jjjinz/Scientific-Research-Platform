个人私有版

通过 CloudBase 自定义登录票据控制访问，只让你自己进入后台。

云端同步

文档、图片与实验附件上传到腾讯云对象存储，跨设备保持同一份数据。

AI 能力

通义千问负责总结与问答，`qwen-vl-ocr` 用于实验图片和扫描图识别。

Setup

当前接入方式

页面默认支持本地模式回退。配置好 `config.js`、CloudBase 环境和云函数后，会自动切换到云端私有版。

1. CloudBase

创建环境、集合和存储桶，并开启 HTTP 访问服务用于登录票据。

2. 通义千问

在阿里云百炼创建 API Key，配置到云函数环境变量，不直接暴露在前端。

3. 前端配置

复制 `config.example.js` 为 `config.js`，填入你的环境 ID、区域和函数入口地址。

Private Access

私有登录

CloudBase 配好后，这里会使用单管理员密码换取登录票据。未配置时会自动回退到本地模式。

Workspace

科研记录与编辑区

想法优先展示，实验和仿真结果支持 AI 总结，文献区支持阅读笔记与图片 OCR。接入云端后，所有内容都将以你的私有账号身份读写。

Ideas

想法总览

持续保存问题定义、方法假设和下一步研究计划。

Simulation

仿真结果

上传图、数据和日志，并生成结构化总结。

Experiment

实验结果

支持附件上传、AI 总结和图片 OCR 识别。

Library

文献库

保存论文、扫描页、阅读笔记，并对图片做 OCR 抽取。
